# Contributing

## Development

### Running Locally

```shell
jekyll serve --config _config.yml,_config.development.yml
# or
jekyll serve --baseurl ""
jekyll serve --baseurl=""
```

## Production

The website will be built and deployed automatically by GitHub.


## Create symbolic link

```shell
ln -s [SOURCE] [TARGET]
ln -s Docs/ _posts
ln -s old_posts/hello.md hello.md
```
